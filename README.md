# terminal-calculator
Compile with:
g++ calc.cpp -o calc

Usage:
./calc '(4*8)^2'

If there are no asterisks, parenthesis, or spaces you don't need quotes:
./calc 4x8

With no arguements you can input the equation without quotes:
./calc
(4*7)^3

x and * for multiplication, / for division, ^ for exponents.
